# Talent Supply & Top Roles Analysis (Power BI)

## 📊 Project Overview
This Power BI dashboard analyzes talent supply distribution across ETM Platform verticals and identifies the top roles within each vertical.

The report supports workforce planning by providing visibility into role concentration and talent allocation across business areas.

---

## ❓ Problem Statement
Recruitment and workforce data was available at a granular level, but lacked:
- Clear visibility of talent supply by vertical
- Identification of dominant roles per business area
- Actionable summaries for planning and decision-making

---

## ✅ Solution
I developed a multi-page Power BI report that:
- Aggregates talent supply by business vertical
- Ranks and displays the top 3 roles per vertical
- Enables easy comparison across business functions

---

## 🧠 Key Features
- Talent supply distribution by vertical
- Top 3 roles per vertical visualizations
- Clean, business-focused dashboard design
- Multi-page report structure

---

## 🛠 Tools & Technologies
- Power BI Desktop
- DAX calculations
- Data modeling
- Workforce analytics
